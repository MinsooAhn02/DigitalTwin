   Replace the single-frame description (lines 477–484) with the warm-up / clean-plate /                                                                                  ↓
   commit-once / lock lifecycle:
   - On camera switch: if a saved pose exists in camera_pose.json → load + lock immediately
   (no warm-up). Otherwise enter warm-up.                                                                                                                                 ↓
   - Warm-up: sample frames at CLEANPLATE_SAMPLE_S=0.5 s into a ring buffer (cap
   CLEANPLATE_MAX_FRAMES=60), evaluate every WARMUP_EVAL_EVERY=30 frames, force commit
   at WARMUP_MAX_S=90 s.                                                                                                                                                  ↓
   - Clean plate = per-pixel median over the stack → removes transient vehicles → reveals
   static lane paint → dashed-line autocorrelation clears its gate (dash_obs ≥ DASH_MIN_OBS=3). This is the root fix for dash_obs=0 on a single compressed frame.
   - Commit: upscale clean plate, run auto_calibrate_from_frame, save pose, fit scale model                                                                               ↓
   once, set _locked. After lock the vehicle-scale refit is frozen and the pose never
   re-solves. POST /recalibrate clears the lock and restarts warm-up.
                                                                                                                                                                          ↓
   Then extend the pose-solver paragraph (lines ~486–508) with the focal-free solve:
   - Default fixed-focal solve: θ=(H, pitch, yaw, x0), focal fixed at FOCAL_RATIO·h,
   scipy.optimize.least_squares (soft-L1) over 5 rows of lane-edge reprojection + a VP                                                                                    ↓
   residual; gated by residual < POSE_RESIDUAL_MAX_PX=8 px.
   - Focal-free solve: when dash_obs ≥ FOCAL_FREE_MIN_OBS=3 (and the depth-span gate
   FOCAL_FREE_MIN_ROW_FRAC passes), focal becomes a 5th free parameter. The dashed-line
   period anchors the longitudinal scale that road width alone cannot observe — the
   geometric fix for speed overestimation. On the clean plate this gate reliably fires; on a
   single live frame it rarely does (motivating the warm-up accumulator).
   - Add a short description of lane_markings.py (5 strips, column-intensity autocorrelation,
   peak_val ≥ 0.15, period grows with depth).
